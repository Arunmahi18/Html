Defintion list

Web Development Technologies

HTML
HTML stands for Hyper Text Markup Language. It's used to markup a content's meaning with tags.
CSS
CSS stands for Cascading Style Sheet. It's used to style the web page's look and feel.
JS
JS stands for Javascript. It's used to determine the behaviour of web page.
NodeJs
NodeJS is a server side programming language developed with javascript. Though we can't rule them all with one ring, we can code them all in one language.
